# Sparacino Conjecture (2026)

**Description:**  
Verification of prime intervals following \(d \cdot p_1\), where \(d\) is the gap between consecutive primes \(p_1\) and \(p_2\).  

- Computational results: 50,847,534 consecutive prime pairs (\(p_1 \le 10^9\)) verified, no counterexamples found.  
- Interval definition: \([d \cdot p_1 + 1, d \cdot p_1 + \lceil \ln^2(d \cdot p_1) \rceil]\)  
- Methods: Optimized Sieve of Eratosthenes, Miller–Rabin deterministic primality test (64-bit).  
- Code available in Python for reproducibility.  

**Potential significance:**  
Empirical evidence suggests a new regularity of primes. Connections with the Riemann Hypothesis and Cramér’s conjecture are discussed in the preprint.
